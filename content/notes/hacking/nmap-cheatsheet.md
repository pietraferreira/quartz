| **nmap flag**   | **Description**                                            |
| --------------- | ---------------------------------------------------------- |
| `-sV`           | Versions                                                   |
| `-p <x> or -p-` | Port scan for port <x> or all                              |
| `-Pn`           | Disable host discovery                                     |
| `-A`            | Enables OS and version detection, execute built-in scripts |
| `-sC`           | Scan with the default nmap scripts                         |
| `-v`            | Verbose mode                                               |
| `-sU`           | UDP port scan                                              |
| `-sS`           | TCP SYN port scan`                                         |
